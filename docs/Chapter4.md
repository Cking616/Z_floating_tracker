* 直线型斜坡

    ![][slope3]
    ~~~
    斜坡坡度系数 m = b / h， 取 dlta = 1 / m便于下面符号运算，
    一般运动时平面运动分析以v做匀速运动，
    或以a做匀加速运动则有：
    输入： yd(t) = v * dlta * t （匀速运动）
    或：   yd(t) = 1 / 2 * a * dlta * ( t ^ 2) （匀加速运动）
    根据现实条件有：
    dlta < 0.3，a <= 15 m/s^2，v <= 2 m/s；
    宽度b一般也有物理条件限制。
    ~~~
